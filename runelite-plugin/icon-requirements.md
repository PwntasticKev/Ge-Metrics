# Plugin Icon Requirements

## Specifications
- **Size**: 64x64 pixels (exactly)
- **Format**: PNG with transparency
- **Location**: `src/main/resources/icon.png`
- **Content**: GE Metrics logo/branding

## Icon Concept
- **Background**: Transparent or dark blue/purple gradient
- **Main Element**: Gold/yellow coin or Grand Exchange symbol
- **Secondary**: Small graph/chart overlay showing upward trend
- **Text**: "GM" or "GE" initials if space allows
- **Style**: RuneScape-themed, fits with other plugin icons

## Temporary Solution
Currently the plugin loads with a default icon (16x16 placeholder) in GeMetricsPlugin.java lines 58-66.
The plugin will function without the icon, but it's required for Plugin Hub submission.

## Creation Tools
- GIMP (free)
- Photoshop
- Canva (with PNG export)
- Figma
- Any image editor that can export 64x64 PNG with transparency

## Example Icon Elements
1. Gold coin with "GE" text
2. Small upward trending arrow/graph
3. RuneScape-style border or glow effect
4. Colors matching RuneScape UI (golds, browns, blues)
